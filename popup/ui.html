How to use

  1. Open any AI chat site — ChatGPT, Claude, Gemini, or Z.ai.
  2. Find the floating Export / Continue Chat button under the chat input box.
  3. Click Continue Chat to copy the conversation as a ready-to-paste prompt.
  4. Start a new chat anywhere and paste — your conversation continues seamlessly.
